
📄 LEIA-ME - Organizador de Arquivos

🧠 O que é isso?
Esse é um organizador de arquivos automático. Ele vai pegar todos os arquivos da pasta onde ele está e separar em pastas como Imagens, Documentos, Vídeos e Músicas, deixando tudo organizado de forma rápida e prática.

🚀 Como usar?
1. Coloque o arquivo 'Organizador.exe' na pasta que você quer organizar (ex.: Downloads, Documentos, Área de Trabalho, etc).
2. Dê dois cliques no arquivo 'Organizador.exe'.
3. Ele vai criar pastas como:
   - 🖼️ Imagens
   - 📄 Documentos
   - 🎥 Vídeos
   - 🎵 Músicas
4. Todos os arquivos serão movidos automaticamente para suas respectivas pastas.

⚠️ Atenção!
- Esse programa move os arquivos, então eles saem do local original e vão para as pastas organizadas dentro dessa mesma pasta.
- Verifique se é essa a pasta que você quer organizar antes de executar.
- Caso não queira mais, é só excluir as pastas criadas — os arquivos estarão lá dentro.
